linkAuthor Information

You can add author information to each page using :Author component:

1> :Author name=Jackie Chan, \

2> avatar='https://m.media-amazon.com/images/M/MV5BMTk4MDM0MDUzM15BMl5BanBnXkFtZTcwOTI4MzU1Mw@@._V1_UY1200_CR117,0,630,1200_AL_.jpg', \

3> date=Yesterday, \

4> url=https://en.wikipedia.org/wiki/Jackie_Chan



warning IMPORTANT

Remember to remove all of these boilerplate pages before you publish your blog!

linkGetting Info from GitHub

If your blog is hosted on GitHub, you can also fetch author info automatically using commit logs. Simply configure your repo via .codedoc/config.ts:

.codedoc/config.ts
1

2import { configuration, 

3  DefaultMarkdownCustomComponents,

4} from '@codedoc/core';

5

6// ...

7

8

9export const config = /*#__PURE__*/configuration({

10  // ...

11  misc: {

12    github: {

13      repo: 'my-cool-blog',               // --> your repo's name

14      user: 'john-doe'                    // --> your repo's user name

15    }

16  }

17});


And then just pass src=github to :Author component:

1> :Author src=github
